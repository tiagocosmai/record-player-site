📸 COMO TIRAR SCREENSHOTS:

   WINDOWS:
   - Pressione: Print Screen (PrtSc)
   - Abra Paint
   - Ctrl+V (colar)
   - Salvar como PNG

   MAC:
   - Pressione: Cmd + Shift + 4
   - Selecione área
   - Arquivo salvo na área de trabalho

   LINUX:
   - Pressione: Print Screen
   - Ou use ferramenta Captura de Tela


🖼️ CONVERTER SVG PARA PNG:

   MÉTODO FÁCIL:
   1. Arraste arquivo .svg para navegador
   2. Botão direito → "Salvar imagem como"
   3. Escolha PNG
   4. Pronto!

   ONLINE:
   - CloudConvert.com (gratuito)
   - Photopea.com (editor online)
